Mapalengke

Project information

  • Category: Mobile Application / GIS Development
  • Type of Project: Capstone Project
  • Project date: 2024 - 2025
  • Project URL: https://mapalengke.me/
  • Project URL: https://github.com/hbralcaide/mobile

A GIS-Based Digital Stall Mapping and Navigation System is a mobile application developed for Toril Public Market that modernizes indoor navigation and stall information access using GIS technology, interactive indoor maps, and real-time updates. Built with React Native, Supabase, PostGIS, and the Mappedin SDK, the app allows shoppers to easily locate stalls, view vendor details, and browse available products, while vendors can instantly update their stall data through Supabase Realtime. Designed to enhance market accessibility and support digital transformation, this project demonstrates practical use of indoor mapping, geospatial databases, and mobile development in a real-world environment.
